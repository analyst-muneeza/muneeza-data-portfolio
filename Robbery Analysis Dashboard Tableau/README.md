# Robbery Analysis Dashboard

Interactive Tableau Dashboard created to analyze robbery trends, crime distribution, and jurisdiction-wise crime statistics.

## Tools Used
- Tableau Public
- Data Visualization

## Dataset Used
- `report.xlsx`

## Dashboard Features
- Jurisdiction-wise Robbery Analysis
- Year-wise Crime Trend Analysis
- Robberies vs Rapes Comparison
- Multi-Crime Category Analysis
- Area Chart Visualization
- Dual Axis Charts
- Lollipop Chart Analysis
- Interactive Crime Insights

## Key Insights
- Certain jurisdictions reported significantly higher robbery cases compared to others.
- Robbery trends peaked during specific years and gradually declined afterward.
- Comparative analysis highlights relationships between robberies and other crime categories.
- Multi-crime visualization helps understand overall crime distribution patterns.

## Visualizations Included
- Bar Chart
- Line Chart
- Area Chart
- Dual Axis Charts
- Combined Axis Chart
- Lollipop Chart

## Files Included
- `Robbery report.twbx` → Tableau dashboard file
- `report.xlsx` → Dataset used for analysis
