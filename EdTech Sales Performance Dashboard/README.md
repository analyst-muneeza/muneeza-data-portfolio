# 🎓 EdTech Sales Performance Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes lead generation, conversion performance, counselor efficiency, and city-wise revenue trends for an EdTech business.

## 🔧 Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## 📊 Key Insights
- Generated 83M revenue from 10K leads.
- Conversion rate remained stable at 15.22%.
- Instagram & YouTube delivered the highest conversions.
- Higher counselor engagement improved conversions.

## 🚀 Features
- KPI Cards
- Funnel Analysis
- Conversion Analysis
- City-wise Revenue Map
- Counselor Performance Tracking
- Interactive Slicers
