# COVID-19 Trends and Forecast Dashboard – India

## Project Overview
This Power BI dashboard analyzes COVID-19 trends in India, including vaccination progress, rolling average cases, month-over-month changes, and 30-day forecasting.

## Tools Used
- Power BI
- DAX

## Key KPIs
- Total Vaccinations: 2B+
- Rolling 7-Day Average Tracking
- MoM % Change Analysis
- 30-Day Forecasting

## Features
- COVID-19 Trend Analysis
- Vaccination Progress Tracking
- Forecasting Visualization
- Month-over-Month Growth Analysis
- Interactive Dashboard Design

## Key Insights
- India crossed 2 billion vaccination doses.
- COVID cases show major peaks followed by decline.
- Month-over-Month growth remains mostly negative.
- Forecast suggests stable low case numbers.

## Files Included
- Power BI Dashboard (.pbix)
- Dashboard Screenshots
- Summary & Insights
