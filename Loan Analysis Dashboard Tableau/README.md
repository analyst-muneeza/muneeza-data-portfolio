# Loan Analysis Dashboard

Interactive multi-dashboard Loan Analysis project created using Tableau Public.

## Tools Used
- Tableau Public
- Data Visualization

## Dashboards Included
- Demographic Analysis
- Financial & Loan Insights
- Loan Decision Analysis

## Key Insights
- Male applicants dominate loan applications.
- Married and graduate applicants show higher participation.
- Credit history strongly affects loan approval.
- Income and loan amount trends help understand applicant behavior.

## Files Included
- `Loan_Analysis_Dashboard.twbx` → Tableau dashboard file
- `loan_approval_dataset.csv` → Dataset used for analysis
