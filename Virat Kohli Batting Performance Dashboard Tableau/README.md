# Virat Kohli Batting Performance Dashboard

Interactive Tableau Dashboard analyzing Virat Kohli’s batting performance using cricket statistics data from the Batsman dataset.

## Tools Used
- Tableau Public
- Data Visualization

## Dataset Used
- `Batsman.xlsx`

## Dashboard Features
- Opponent-wise Runs Analysis
- Runs vs Balls Faced Comparison
- Batting Performance Trends
- Country-wise Performance Map
- Average Runs Analysis
- Interactive Filters and Visualizations
- Performance Contribution Analysis

## Key Insights
- Virat Kohli scored the highest runs against Sri Lanka and West Indies.
- Strong batting consistency is observed across major cricketing nations.
- Runs scored and balls faced show a strong positive relationship.
- Performance analysis highlights opponent-wise batting efficiency.

## Visualizations Included
- Bar Charts
- Dual Axis Chart
- Scatter Plot
- Funnel Chart
- Pie Chart
- Packed Bubble Chart
- Map Visualization
- Interactive Filters

## Files Included
- `Virat Kohli – Batting Performance Dashboard.twbx` → Tableau dashboard file
- `Batsman.xlsx` → Dataset used for analysis
