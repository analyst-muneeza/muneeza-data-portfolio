# SuperStore Dashboard

Interactive SuperStore Sales Dashboard created using Tableau Public to analyze sales performance, profit trends, and customer insights.

## Tools Used
- Tableau Public
- Data Visualization

## Dashboard Features
- Sales & Profit Analysis
- Regional Performance Insights
- Category & Sub-Category Analysis
- Customer Segment Analysis
- Order & Shipping Trends
- Interactive Filters and Visualizations

## Key Insights
- Technology category generated the highest sales.
- Certain sub-categories showed high sales but low profit margins.
- Regional analysis highlights top-performing areas.
- Customer segment analysis helps understand purchasing behavior.

## Files Included
- `SuperStore_Dashboard.twbx` → Tableau dashboard file
- `Load data from Excel- Superstore.xls` → Dataset used for analysis
