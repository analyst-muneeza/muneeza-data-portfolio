# COVID-19 Dashboard

Interactive COVID-19 Dashboard created using Tableau Public using WHO global COVID-19 dataset.

## Tools Used
- Tableau Public
- Data Visualization

## Dashboard Features
- Global COVID-19 Case Analysis
- Country-wise Confirmed Cases
- Monthly Confirmed Cases Trend
- Interactive World Map Visualization
- Top Affected Countries Analysis

## Key Insights
- The United States reported the highest number of confirmed cases.
- COVID-19 cases increased rapidly during major outbreak periods.
- Monthly trend analysis helps identify peak infection periods.
- Interactive visualizations provide better understanding of global spread patterns.

## Files Included
- `COVID-19_Dashboard.twbx` → Tableau dashboard file
- `WHO-COVID-19-global-data.csv` → Dataset used for analysis
