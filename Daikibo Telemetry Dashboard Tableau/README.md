# Daikibo Telemetry Dashboard

Interactive Machine Downtime Analysis Dashboard created using Tableau Public to monitor factory performance and device downtime trends.

## Tools Used
- Tableau Public
- Data Visualization

## Dashboard Features
- Factory-wise Downtime Analysis
- Device Type Downtime Monitoring
- Machine Performance Insights
- Comparative Factory Analysis
- Interactive Visualizations

## Key Insights
- Daikibo Factory Seiko reported the highest machine downtime.
- LaserWelder and LaserCutter devices experienced maximum downtime.
- Some device types showed minimal or zero downtime, indicating stable performance.
- Factory comparison helps identify operational inefficiencies.
