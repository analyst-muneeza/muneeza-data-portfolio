## Hospital Emergency Room Dashboard (Excel)

This project presents an interactive Excel dashboard analyzing hospital emergency room operations and patient flow.

---

##  Overview

The dashboard helps hospital administrators and analysts track key performance indicators such as patient volume, waiting times, satisfaction levels, and departmental referrals.

---

## Key Metrics

- Total patients  
- Average wait time  
- Satisfaction score  
- Admission status  
- Gender distribution  
- Age group analysis  
- Department referrals  
- Delay vs On-time status  

---

##  Business Questions Answered

- Which months have the highest ER traffic?
- What percentage of patients are admitted?
- Which departments receive the most referrals?
- Which age groups dominate ER visits?
- How does waiting time impact satisfaction?
- Are delays increasing during peak periods?

##  How to Use

1. Download the Excel file.  
2. Open in Microsoft Excel (desktop version recommended).  
3. Use slicers to filter by year and month.  
4. Explore trends across demographics and departments.
