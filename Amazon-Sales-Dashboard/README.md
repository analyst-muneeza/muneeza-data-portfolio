# 📊 Amazon Sales Dashboard

## 📌 Project Overview
This Power BI dashboard provides an interactive analysis of Amazon sales performance across product categories, products, customer locations, payment methods, and order status. It helps identify sales trends, customer purchasing behavior, and operational opportunities through data-driven insights.

## 🛠️ Tools Used
- Power BI
- DAX

## 📈 Key KPIs
- 💰 Total Sales: **244K**
- 📦 Total Quantity Sold: **714**
- 🛍️ Product Count: **250**
- 📄 Total Orders: **250**

## ✨ Dashboard Features
- Sales by Category
- Top Products by Sales
- Payment Method Analysis
- Customer Location Analysis
- Order Status Monitoring
- Interactive Filters & Slicers

## 💡 Key Insights
- Electronics and Home Appliances generated the highest revenue.
- Refrigerators, Laptops, and Smartphones were the top-selling products.
- PayPal and Credit Card were the most preferred payment methods.
- Miami and Houston recorded the highest sales.
- Around **30%** of orders remain pending, highlighting an opportunity to improve order fulfillment efficiency.
